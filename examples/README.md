# Examples

This directory contains external example/reference projects used by BinaryKeyboard.

## OpenWCH CH592 SDK

- Path: `examples/openwch-ch592`
- Source: `https://github.com/openwch/ch592`
- Type: Git submodule

Clone with submodules:

```bash
git clone --recurse-submodules <repo-url>
```

If already cloned:

```bash
git submodule update --init --recursive
```
