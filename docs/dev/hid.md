# HID 协议与 Report ID

BinaryKeyboard 的通信基于 HID，常见 Report ID 用途如下（示例）：

| Report ID | 功能 | 数据长度 | 说明 |
|---:|---|---:|---|
| 1 | 键盘 | 8 字节 | 模拟键盘输入 |
| 2 | 控制器 | 16 位 * 4 | 模拟控制器输入 |
| 3 | 鼠标 | 5 字节 | 模拟鼠标输入 |
| 4 | 主机 → 键盘 | 31 字节 | 主机发送数据到键盘 |
| 5 | 键盘 → 主机 | 31 字节 | 键盘发送数据到主机 |

> 如果你要做二次开发（例如写自己的配置器/桌面客户端），这是最关键的协议入口。
