# 经典版改键软件

> 现在与无线版共用同一套 BinaryKeyboard Studio，浏览器会自动识别 CH552G / CH592F 固件并切换可用功能。

## 浏览器要求

| 浏览器  | 版本要求 | 支持情况   | 说明              |
| :------ | :------- | :--------- | :---------------- |
| Chrome  | ≥ 89     | ✅ 完全支持 | Chromium 内核     |
| Edge    | ≥ 89     | ✅ 完全支持 | Chromium 内核     |
| Opera   | ≥ 75     | ✅ 完全支持 | 基于 Chromium     |
| Firefox | -        | ❌ 不支持   | WebHID API 未实现 |
| Safari  | -        | ❌ 不支持   | WebHID API 未实现 |

::: warning 重要
本工具依赖 WebHID API，仅支持 Chromium 内核浏览器（Chrome、Edge、Opera 等）。
:::

## 使用步骤

### 1. 连接设备

1. 用 USB-C 连接键盘到电脑
2. 打开改键工具页面
3. 点击"连接设备"按钮
4. 在弹出的设备选择框中选择你的键盘
5. 授权浏览器访问 USB 设备

### 2. 配置键位

1. 选择要修改的源键位
2. 选择映射类型：
   - 🎹 键盘键位
   - 🎵 多媒体键
   - 🖱️ 鼠标键
3. 设置目标键位

::: tip CH552G 功能范围
经典版（CH552G）当前支持：

- 单层键位映射
- 键盘 / 多媒体 / 鼠标动作
- 基础款、五键款、旋钮款布局自动识别

当前不支持：

- 多层切换配置
- FN 键配置
- RGB 灯效配置
- 电池 / HID 日志面板
:::

### 3. 写入设备

1. 确认配置无误
2. 点击"写入键位"
3. 等待写入完成

::: tip
配置写入后会保存在键盘内部，拔掉 USB 也不会丢失。
:::

## 常见问题

### 无法连接设备？

1. 确保使用 Chrome 或 Edge 浏览器
2. 确保网站使用 HTTPS（或 localhost）
3. 检查键盘是否正确连接
4. 检查是否有其他程序占用设备

### 如何恢复默认键位？

重新刷写固件，并在下载配置中选择“清空 DataFlash”。
