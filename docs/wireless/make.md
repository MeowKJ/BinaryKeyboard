## 与经典版的差异

| 项目     | 经典版   | 无线版          |
| :------- | :------- | :-------------- |
| 主控芯片 | CH552G   | CH592F          |
| 连接方式 | USB      | USB、蓝牙、2.4G |
| 供电     | USB 供电 | 锂电池          |
| 充电     | -        | USB-C 充电      |
| 外壳     | 标准     | 需电池仓        |

## 焊接要点

### CH592F 芯片

- 注意芯片方向，对准丝印小圆点
- CH592F 引脚更密，焊接时需要小心
- 焊接后仔细检查虚焊或桥连

### 电池接口

- **务必确认正负极**
- 确认为3.7V锂电池
  
### 天线

- 天线区域不要搞焊锡上去
