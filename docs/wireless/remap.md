# 无线版改键软件

> 无线版同样支持 Web 改键工具，但需要通过 USB 连接进行配置

## 浏览器要求

| 浏览器  | 版本要求 | 支持   |
| :------ | :------- | :----- |
| Chrome  | ≥ 89     | ✅ 推荐 |
| Edge    | ≥ 89     | ✅ 推荐 |
| Opera   | ≥ 75     | ✅      |
| Firefox | -        | ❌      |
| Safari  | -        | ❌      |

::: warning 重要
改键需要通过 USB-C 连接，蓝牙、2.4G 模式下暂时无法进行配置。
:::

## 使用步骤

### 1. 切换到 USB 模式

1. 用 USB-C 连接键盘到电脑
2. 键盘会自动切换到 USB 模式
3. 此时蓝牙连接会断开

### 2. 连接改键工具

1. 打开改键工具页面
2. 点击"连接设备"按钮
3. 在弹出的设备选择框中选择你的键盘
4. 授权浏览器访问 USB 设备
