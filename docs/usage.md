# 使用指南

这页按实际使用顺序整理 BinaryKeyboard：先选对型号和固件，再连接、改键、切层、使用 RGB / 宏，最后再进入开发文档。

## 型号速查

| 系列 | 芯片 | 发布型号 | 连接方式 | 适合场景 |
| :--- | :--- | :--- | :--- | :--- |
| 经典版 | CH552G | BASIC / 5KEY / KNOB | USB 有线 | 简单、稳定、即插即用 |
| 无线版 | CH592F | 5KEY / KNOB | USB / BLE | 需要无线、宏、FN、状态指示 |

::: warning 先确认型号
固件按型号区分，BASIC、5KEY、KNOB 不能混刷。刷错后最常见的现象是按键位置不对、旋钮无反应或 Studio 显示布局异常。
:::

## 第一次使用

1. 从 [GitHub Releases](https://github.com/MeowKJ/BinaryKeyboard/releases) 下载对应型号固件。
2. 按 [经典版刷写](./wired/flash.md) 或 [无线版刷写](./wireless/flash.md) 完成首次烧录。
3. 用 Chrome / Edge 打开 BinaryKeyboard Studio。
4. USB 连接键盘，点击“连接设备”，授权 WebHID。
5. 按需要修改键位、层、RGB、宏，然后写入设备。

::: tip 浏览器
Studio 使用 WebHID，只支持 Chrome、Edge、Opera 等 Chromium 内核浏览器。Firefox 和 Safari 当前不可用。
:::

## 通用概念

### 键位编号

Studio 里的键位编号与固件中的索引一致，从 1 开始显示。文档里写的 K1、K2、K3 等，对应 Studio 中从左到右、从上到下看到的可点击键位。

### 层

层可以理解成多套键位配置。当前层切换后，同一个实体按键会执行当前层里的动作。

- 经典版：BASIC / KNOB 默认 4 层，5KEY 默认 5 层。
- 无线版：KNOB 默认 4 层，5KEY 默认 5 层。
- Studio 中可逐层编辑，每层写入后保存在设备内。

### 可配置动作

| 动作 | 经典版 CH552G | 无线版 CH592F |
| :--- | :--- | :--- |
| 键盘按键 | 支持 | 支持 |
| 修饰键组合 | 支持 | 支持 |
| 鼠标按键 / 滚轮 | 支持 | 支持 |
| 多媒体键 | 支持 | 支持 |
| 层动作 | 部分支持，以 Studio 当前可见项为准 | 支持 |
| 宏 | 支持 MeowFS 宏存储 | 支持 MeowFS 宏存储 |
| RGB 配置 | 支持按键 RGB | 支持按键 RGB / 状态指示 |
| 电池状态 | 不适用 | 支持 |
| HID 日志 | 不适用 | 支持 |

## 经典版 BASIC

BASIC 是 4 键 USB 直连版本，默认布局为 3 列 2 行，其中右侧是一个竖向 2u 键，左下是一个横向 2u 键。

默认键位：

| 键位 | 默认动作 |
| :--- | :--- |
| K1 | `0` |
| K2 | `1` |
| K3 | `Enter` |
| K4 | `Space` |

使用方式：

- 插入 USB 后即可作为标准键盘使用。
- 短按 `FUNC`：切换下一个 RGB 灯效。
- 按住 `FUNC` 再按 K1~K4：切换到第 1~4 层。
- 按住 `FUNC` 时普通按键不会向电脑发送，避免切层时误触发。
- 在 Studio 中选择不同层后，可以分别配置 K1~K4 的动作。

## 经典版 5KEY

5KEY 是 5 键 USB 直连版本，比 BASIC 多一个普通按键，默认 5 层。

默认键位：

| 键位 | 默认动作 |
| :--- | :--- |
| K1 | `1` |
| K2 | `2` |
| K3 | `3` |
| K4 | `4` |
| K5 | `5` |

使用方式：

- 插入 USB 后即可使用。
- 短按 `FUNC`：切换下一个 RGB 灯效。
- 按住 `FUNC` 再按 K1~K5：切换到第 1~5 层。
- 适合把五个键设置成快捷键、媒体控制、应用启动宏或游戏快捷操作。

## 经典版 KNOB

KNOB 是 4 个普通键加 1 个旋钮的 USB 版本。旋钮在 Studio 中会显示为 3 个虚拟动作：按下、左转、右转。

默认键位：

| 键位 | 默认动作 |
| :--- | :--- |
| K1 | `A` |
| K2 | `B` |
| K3 | `C` |
| K4 | `D` |
| 旋钮按下 | 鼠标左键 |
| 旋钮左转 | 音量减 |
| 旋钮右转 | 音量加 |

使用方式：

- K1~K4 像普通键盘按键一样使用。
- 旋钮旋转会触发一次离散动作，适合音量、滚轮、上一首/下一首等。
- 短按 `FUNC`：切换下一个 RGB 灯效。
- 按住 `FUNC` 再按 K1~K4：切换到第 1~4 层；旋钮不参与层选择。
- Studio 中可分别配置 K1~K4、旋钮按下、旋钮左转、旋钮右转。

## 无线版 5KEY

无线 5KEY 是 5 键 USB / BLE 双模版本，默认 5 层。

默认键位：

| 键位 | 默认动作 |
| :--- | :--- |
| K1 | `1` |
| K2 | `2` |
| K3 | `3` |
| K4 | `4` |
| K5 | `5` |

使用方式：

- USB 模式下，连接 USB-C 后作为有线键盘使用，并可打开 Studio 配置。
- BLE 模式下，键盘作为蓝牙 HID 键盘使用。
- `FN1` 短按 / 长按：在 USB 与 BLE 模式之间切换，设备会保存模式并重启进入目标模式。
- `FN2` 短按：切换到下一层。
- `FN2` 长按：在 BLE 模式下清除蓝牙配对信息并重启。
- 按住任意 FN 键再按 K1~K5：直接切换到第 1~5 层。

## 无线版 KNOB

无线 KNOB 是 4 个普通键加 1 个旋钮的 USB / BLE 双模版本。旋钮动作在 Studio 中同样作为虚拟键位配置。

默认键位：

| 键位 | 默认动作 |
| :--- | :--- |
| K1 | `1` |
| K2 | `2` |
| K3 | `3` |
| K4 | `4` |
| 旋钮右转 | 音量加 |
| 旋钮左转 | 音量减 |
| 旋钮按下 | 静音 |

使用方式：

- USB 模式下可直接作为有线键盘使用，也用于 Studio 改键和固件更新。
- BLE 模式下用于无线输入。
- `FN1` 短按 / 长按：切换 USB / BLE 模式。
- `FN2` 短按：切换到下一层。
- `FN2` 长按：在 BLE 模式下清除蓝牙配对信息。
- 按住任意 FN 键再按 K1~K4：切换到第 1~4 层；旋钮不参与层选择。
- 旋钮适合配置为音量、滚轮、媒体键、层切换或宏触发。

## 无线版蓝牙配对

1. 确保电池已连接且电量足够。
2. 短按 `FN1` 切换到 BLE 模式，键盘会重启并进入蓝牙路径。
3. 等待指示灯进入广播状态。
4. 在电脑或手机蓝牙设置中搜索 `BinaryKeyboard5KEY` 或 `BinaryKeyboardKNOB`。
5. 点击连接，按系统提示完成配对。

如果系统一直连不上：

- 在 BLE 模式下长按 `FN2` 清除配对，再重新搜索。
- 在电脑 / 手机蓝牙列表中删除旧的 BinaryKeyboard 设备后重试。
- 用 USB 连接 Studio，打开 Debug Terminal 查看 BLE 日志。

## Studio 改键

### 连接

1. 用 USB-C 连接键盘。
2. 打开 Studio。
3. 点击“连接设备”。
4. 在浏览器弹窗中选择 BinaryKeyboard。

::: warning 无线版也要接 USB
无线版改键、宏编辑和固件更新都通过 USB HID 配置通道完成。BLE 模式只负责日常输入，暂不用于配置。
:::

### 写入配置

1. 选择要编辑的层。
2. 点击键位，选择键盘、鼠标、媒体、宏或层动作。
3. 无线版可继续调整 FN、RGB、电池和日志相关配置。
4. 点击保存 / 写入，等待提示成功。

### 宏

宏适合把一串输入绑定到单个键位，例如固定短语、快捷键序列或鼠标点击序列。

- 详细语法见 [MeowMacro](./meowmacro.md)。
- 存储结构见 [MeowFS](./meowfs.md)。
- 无线版宏编辑器说明见 [无线版改键软件](./wireless/remap.md#宏编辑器使用指南)。

## RGB 与状态灯

经典版：

- 短按 `FUNC` 切换灯效。
- Studio 中可配置开关、模式、亮度、速度、颜色和按键反馈效果。

无线版：

- 默认使用状态指示模式。
- USB 已连接、BLE 广播、BLE 已连接、低电量、充电和睡眠都有不同状态提示。
- Studio 中可调整 RGB 和指示灯亮度。

## 进入刷写 / 恢复

经典版：

- 拔下 USB。
- 按住 `BOOT`。
- 插入 USB。
- 松开 `BOOT` 后刷写对应 `CH552G-<MODEL>-<version>.hex`。

无线版：

- 首刷或救砖使用 `CH592F-<MODEL>-<version>-full.hex`。
- 正常在线更新使用 Studio 下载并写入 `CH592F-<MODEL>-<version>-app.bin`。
- 设备正常运行时，单击 `BOOT` 会进入 IAP / Bootloader 路径。
