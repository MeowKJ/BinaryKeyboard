# 快速开始

下面以“拿到 PCB + 外壳文件后从零复刻”为路径，给出最短流程。

## 1. 选择你的版本

### 有线版
- **基础款 / 五键款 / 旋钮款**：三种外壳与固件不同。
- 如果你不确定：先做 **基础款** 或 **五键款**。
  
### 无线版
- **基础款 / 五键款 / 旋钮款**：三种外壳与固件不同。
- 如果你不确定：先做 **基础款** 或 **五键款**。

## 2. 下载文件

- 固件：GitHub Releases 中下载与你版本对应的 `.hex`
- 外壳：OSHWHub 附件或 Releases 中的 `.stl`

## 3. 3D 打印外壳

普通 FDM 打印即可（打印参数按你机器习惯设置）。  

## 4. 焊接 & 组装

- 焊好 PCB 上的核心器件与接口
- 安装轴体/按键（如有热插拔座请先焊好）
- 合入外壳

## 5. 刷写固件

按 [刷写固件](/guide/flash) 操作。

## 6. 打开改键工具并配置键位

使用 Chrome 或 Edge 打开改键工具页面，完成键位映射：  
见 [改键软件使用](/guide/remap)。
