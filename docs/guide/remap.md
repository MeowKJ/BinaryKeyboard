# 改键软件使用（Web 版）

## 功能概览

改键工具支持：

- 键盘键位映射
- 多媒体键映射
- 鼠标键位映射

并可自动识别多款硬件，读取/写入映射。

## 在线使用

使用 **Chrome 或 Edge** 打开在线改键页面即可配置。

> 如果在线地址不可用，你可以选择自己部署（见下）。

## 自部署（建议方式）

仓库中通常会包含 Web 改键工具的源码（Vue3），也可能提供已编译的 Web 包（例如 zip）。

通用部署思路：

1. 下载/解压 web 构建产物
2. 用任意静态文件服务器托管（nginx、Vercel、GitHub Pages 等）
3. 通过浏览器打开页面并连接键盘设备进行改键

> 由于浏览器对 HID/USB 访问权限限制，推荐使用 Chromium 内核浏览器（Chrome/Edge）。
