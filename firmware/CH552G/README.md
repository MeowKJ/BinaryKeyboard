# CH552G 固件

经典版（USB 有线）固件，基于 **CH552G** 8051 芯片。

## 文档

完整开发文档：[经典版开发指南](../../docs/wired/dev.md)

## 快速开始

1. 安装 Arduino IDE 及 [CH55xduino](https://github.com/DeqingSun/ch55xduino) 库
2. 打开 `CH552G.ino`，选择板子 **CH552**，USB Setting 选 **USER CODE w/148B USB ram**
3. 按住 **BOOT** 按钮连接 USB 进入 Bootloader，点击上传
